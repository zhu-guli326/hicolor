# hicolor — 创意文字散落/打孔工具

## 概述

hicolor 是一款基于 Web 的创意图片处理工具，核心功能是：**用英文字母（A-Z）或自定义符号在图片上"挖洞"，然后在图片旁边拼接一块彩色色块，透过字母形状的空洞露出背后的彩色区域**，形成一种文字打孔/散落的视觉创意效果。

> 项目地址（AI Studio）：https://ai.studio.apps/e5b9fce5-b82e-4e8c-a602-2838d1830b6e

## 核心功能

### 图片上传
- 支持 JPG、PNG、WEBP 格式
- 上传后自动根据图片横竖选择布局方向（竖图 → 色块在右侧；横图 → 色块在下方）

### 背景色块
- **纯色模式**：单色填充
- **条纹模式**：双色条纹，可调节条纹粗细
- **4种排版方向**：色块在上方 / 下方 / 左侧 / 右侧

### 取色器
- 点击取色按钮后，在图片上任意位置点击吸取颜色
- 可分别吸取条纹的两种颜色

### 元素（字母/符号）生成
- **自动生成模式**：按指定数量在图片边缘区域随机分布，支持边缘偏好（更集中在四角/边缘）
- **手动点击模式**：在画布上任意点击放置元素
- **分布模式**：对称分布 或 打散分布
- 每次重新生成字母和位置都会随机变化

### 元素配置
| 参数 | 说明 |
|---|---|
| 基础大小 | 字母的基准像素尺寸 |
| 随机差异 | 字母大小在基准上的随机波动幅度 |
| 旋转角度 | 每个字母的随机旋转角度（0°–360°） |
| 文字内容 | 随机 A-Z，或自定义单个/多个字符（★ ♥ ● 等符号也支持） |

### 导出
- 将图片与色块按当前布局合成为完整 PNG 文件并下载

## 使用流程

1. **上传图片** — 点击上传按钮或拖拽图片
2. **自动生成** — 系统在图片边缘区域生成一批字母形状的空洞
3. **预览效果** — 图片旁边拼接条纹色块，透过字母空洞露出彩色区域
4. **调整参数** — 切换布局方向、修改颜色、调整字母样式与分布模式
5. **导出保存** — 点击「保存作品」下载最终 PNG
